# XingMinRoad-Services

兴民路服务

## 1. 技术选型

* Windows注册服务工具： https://nssm.cc/
* 后端web框架： https://fastapi.tiangolo.com/

## 2. 目录说明

* data 里存放 .sql文件

* docs 里存放一些相关的资料及部署的文档

* static 里存放前端的代码 和 fastapi的接口文档静态文件

* src 里存放源码

* test 里存放一些测试脚本

* logs 里存放日志文件（保留10天,settings中可以设置）
